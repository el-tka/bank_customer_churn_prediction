# Bank Customer Churn Prediction

Machine learning model that predicts whether a customer will leave the bank.

Customer retention is significantly cheaper than acquiring new customers, so predicting churn allows banks to proactively retain valuable clients.

---

## Dataset

The dataset contains information about bank clients:

- Credit Score
- Geography
- Gender
- Age
- Balance
- Number of Products
- Tenure
- Estimated Salary

Target variable:

- Exited (1 = customer left the bank)

---

## Technologies

Python  
Pandas  
Scikit-learn  
Matplotlib  
Seaborn  

---

## Models Tested

- Logistic Regression
- Decision Tree
- Random Forest

The best performance was achieved using **Random Forest**.
